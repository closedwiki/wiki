---+ !AddMetaPlugin

This plugin is designed to be used in template topics, to give them a way to add meta data at the time of topic creation


---++ Syntax Rules

   * %<nop>ADDMETA{type="type" key="key" value="value"}%</verbatim>
   * %<nop>ADDMETA{type="type" value="value"}%</verbatim>

In the latter case, key defaults to =keyh=


---++ Plugin Settings

Plugin settings are stored as preferences variables. To reference
a plugin setting write ==%<nop>&lt;plugin&gt;_&lt;setting&gt;%==, i.e. ==%<nop>ADDMETAPLUGIN_SHORTDESCRIPTION%==

   * One line description, is shown in the %TWIKIWEB%.TextFormattingRules topic:
      * Set SHORTDESCRIPTION = Allows template topics to add (simple) meta data to topics based on them


   * Debug plugin: (See output in =data/debug.txt=)
      * Set DEBUG = 0
