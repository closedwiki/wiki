Cairo compatibility module

---+ Summary of Contents
---+++ <nop>CairoCompatibilityModule
This module implements some of the functions added in Cairo so they are available to plugins running in a pre-Cairo installation. This allows plugin authors to upgrade to Cairo without losing compatibility with earlier TWiki releases.
