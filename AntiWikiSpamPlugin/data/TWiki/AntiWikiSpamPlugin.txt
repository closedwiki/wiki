---++ Plugin Settings

Plugin settings are stored as preferences variables. To reference a plugin setting write ==%<nop>&lt;plugin&gt;_&lt;setting&gt;%==, i.e. ==%<nop>INTERWIKIPLUGIN_SHORTDESCRIPTION%==

   * One line description, is shown in the %TWIKIWEB%.TextFormattingRules topic:
      * Set SHORTDESCRIPTION = prevents registered WikiSpam from being saved

   * Debug plugin: (See output in =data/debug.txt=)
      * Set DEBUG = 1

   * where to get the regex list to check for
      * Set ANTISPAMREGEXLISTURL = http://arch.thinkmo.de/cgi-bin/spam-merge
      * Set ANTISPAMREGEXLISTTOPIC = Plugins.AntiWikiSpamPluginList
   * time in minutes between attempts to get a new version of the list
      * Set GETLISTTIMEOUT = 60
   * Local spam list (added to the common one
      * Set LOCALANTISPAMREGEXLISTTOPIC = Plugins.LocalAntiWikiSpamPluginList
