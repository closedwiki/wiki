---+!! Send Email Plugin

*Allows to send e-mail through an e-mail form.*

%TOC%

---++ Usage
   * Write =%<nop>SENDEMAIL%= anywhere on the page, but preferably in a visible spot because the feedback messages will be written at the tag location.
      * Optionally pass parameters =feedbackSucces= and =feedbackError=. Default feedback messages are set below in [[#PluginSettings][Plugin Settings]].
   * Create a form on the page.
      * Required is a hidden field with =name="sendmail"= and =value="on"=
      * Required is also a field with =name="to"=
      * Optional are the fields =cc=, =subject= and =body=
      * Field =from= may be used but by default the address of =%<nop>WIKIWEBMASTER%= is used (this is set to: %WIKIWEBMASTER%)



---++ Example
---+++ Example form

This form only works if this plugin is enabled and =%<nop>SENDEMAIL%= is written in the topic - see [[#PluginTest][Plugin Tests]] below to check this.

<form enctype="application/x-www-form-urlencoded" name="mailform" action="" method="post">
<input name="sendmail" type="hidden" value="on" />
<p>To:<br /><input class="twikiInputField" id="to" name="to" size="30" type="text" value="" /></p>
<p>CC:<br /><input type="text" class="twikiInputField" id="cc" name="cc" size="30"  value="" /></p>
<p>Subject:<br /><input type="text" class="twikiInputField" id="subject" name="subject" size="30" value="" /></p>
<p>Message:<br /><textarea class="twikiInputField" cols="80" name="body" rows="6"></textarea></p>
<p><input type="submit" class="twikiSubmit" value="Send message" %IF{"context SendEmailPluginEnabled" then='' else='disabled="disabled"'}% /></p>
</form>

#PluginSettings
---++ Plugin Settings

Plugin settings are stored as preferences variables. To reference a plugin setting write ==%<nop>&lt;plugin&gt;_&lt;setting&gt;%==, i.e. ==%<nop>CACHECONTENTPLUGIN_SHORTDESCRIPTION%==

   * One line description, is shown in the %TWIKIWEB%.TextFormattingRules topic:
      * Set SHORTDESCRIPTION = Allows to send e-mail through an e-mail form

   * Default feedback messages:
      * Set EMAIL_SENT_SUCCESS_MESSAGE = Email sent!
      * Set EMAIL_SENT_ERROR_MESSAGE = Could not send email

   * Debug plugin: (See output in =data/debug.txt=)
      * Set DEBUG = 0
