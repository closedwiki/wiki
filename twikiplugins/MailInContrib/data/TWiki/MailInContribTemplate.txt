---+ Templates for the MailInContrib

*DO NOT MODIFY THIS TOPIC*

This is a normal TWiki topic, but it is also a template file, as described in %TWIKIWEB%.TWikiTemplates. It contains the template definitions used by the MailInContrib to insert mail messages in topics.

Each template is specified in the form of a TWiki template definition, and is surrounded by verbatim tags for readability. The template to use when inserting incoming mail in a topic is decided from the parameters in the =&lt;--MAIL--> comment in the topic. The default template is =normal=.

<verbatim>
%TMPL:DEF{MAILIN:normal}%   * *%SUBJECT%*: %TEXT% -- %WIKIUSERNAME% - %SERVERTIME%
%TMPL:END%
</verbatim>

*DO NOT MODIFY THIS TOPIC*

This topic automatically includes a topic called MailInContribUserTemplate, which is *not* shipped with the MailInContrib. Create and modify *that* topic instead.
