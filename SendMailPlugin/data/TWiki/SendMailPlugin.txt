---++ Syntax Rules

=%<nop>SENDMAIL{"..."}%=

| *Parameter* | *Explanation* | *Default* |
| =action=""= | Only =action="send"= is supported, it will send an e-mail | =""= (no action) |
| =from="admin@example.com"= | E-mail address of sender. Defaults to TWiki administrator's e-mail address | ="$webmastername &lt;$webmasteremail&gt;"= |
| =to="jom@example.com"= | To list: Comma-space delimited list of e-mail addresses of adressees. Tokens =$username= and =$useremail= resolve to the WikiName and e-mail address of the logged in user, respectively. Defaults to TWiki administrator's e-mail address | ="$webmastername &lt;$webmasteremail&gt;"= |
| =cc="jimmy@example.com"= | CC list: Comma-space delimited list of e-mail addresses | =""= |
| =bcc="boss@example.com"= | BCC list: Comma-space delimited list of e-mail addresses | =""= |
| =subject="Any text"= | E-mail subject. Text may include variables such as =%<nop>URLPARAM{subject}%=. Format tokens such as =$n= and =$percnt= are expanded | (help message) |
| =text="Any text"= | E-mail body text. Text may include variables such as =%<nop>URLPARAM{subject}%=. Format tokens such as =$n= and =$percnt= are expanded | (help message) |
| =success="..."= | Text shown in place of the SENDMAIL variable on success, default is empty. Format tokens such as =$n= and =$percnt= are expanded | =""= |
| =error="&#124; $error &#124;&#124;"= | Error message shown in place of the SENDMAIL variable on error, if any. Token =$error= is the error message. Format tokens such as =$n= and =$percnt= are expanded. | ="$error"= |

---++ Plugin Installation &amp; Configuration

__Note:__ You do not need to install anything on the browser to use this plugin. The following instructions are for the administrator who installs the plugin on the TWiki server.

   * For an __automated installation__, run the [[%SCRIPTURL{configure}%][configure]] script and follow "Find More Extensions" in the in the __Extensions__ section.

   * Or, follow these __manual installation__ steps:
      * Download the ZIP file from the Plugins home (see below).
      * Unzip ==%TOPIC%.zip== in your twiki installation directory. Content:
        | *File:* | *Description:* |
        | ==data/TWiki/%TOPIC%.txt== | Plugin topic |
        | ==lib/TWiki/Plugins/%TOPIC%.pm== | Plugin Perl module |
      * Set the ownership of the extracted directories and files to the webserver user.
      * Install the dependencies (if any).

   * Plugin __configuration and testing__:
      * Run the [[%SCRIPTURL{configure}%][configure]] script and enable the plugin in the __Plugins__ section.
      * Configure additional plugin settings in the __Extensions__ section.
      * Test if the installation was successful: See example above.
