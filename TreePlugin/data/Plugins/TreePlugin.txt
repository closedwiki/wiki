---++ %TOPIC%

Provides support to view and manage TWiki topic trees (topic hierarchies of parents/children).

_Alpha release. Documentation will be revised. More features finalized._

---+++ Syntax Rules

Adds the following tags:
	* %<nop>TREEVIEW% - displays a TWiki topic's children in a hierarchical fashion.

<blockquote>
Attributes:
	* *web* - specify web
	* *topic* - specify topic
	* *formatting* - specify the formatting method ()
	* *format* - specify the format for each node
	* *formatbranch* - specify the format for a group of children
	* *stoplevel* - the depth of the tree to show
	* *header* - header output
	* *bookview* - list topics in BookView form
</blockquote>

See Plugins.TreePluginSamples for samples.
