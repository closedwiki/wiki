---+!! Personal Info !AddOn

*User personalisation and reference tool.*

A set of elements to enhance user information on the user homepage:
   * User picture
   * Quick overview of the most common user information
   * Quickly edit user information
   * Search tools over this information

When installed, see [[_PersonalInfoAddOn.PersonalInfoHome]] for additional details.

---++ Installation Instructions

   * Download the ZIP file from the Plugin web (see below).
   * Unzip ==%TOPIC%.zip== in your ($TWIKI_ROOT) directory.
   * Copy the files from =/data/_PersonalInfoAddOn= to your user Web (by default Main).
   * !PersonalInfoAddOn is based on TWiki 4.1 with !PatternSkin.
   * Optional plugin: TWiki:Plugins/AttachContentPlugin TWiki:Plugins/AttachContentPlugin
