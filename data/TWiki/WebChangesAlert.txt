---+ Web Changes Notification Service

Each TWiki web has an automatic email notification service that sends you an email with links to all of the topics modified since the last alert. 

%INCLUDE{"MailerContrib"}%

---+ Sysadmins
Notification is supported by an add-on to the TWiki kernel called the MailerContrib. See the MailerContrib topic for details of how to set this service up.
