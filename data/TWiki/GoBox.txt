---+ Go Box
The box at the top or sidebar of each page. 

Enter a topic name to quickly jump to the topic, for example:
	* *WebNotify* to jump to <nop>WebNotify in the current web
	* *%MAINWEB%.WebNotify* to jump to <nop>WebNotify in the %MAINWEB% web
	* *%MAINWEB%.* to jump to the home of the %MAINWEB% web
	* *BrandNewTopic* to jump to a non existing topic in the current web, which is useful to create orphaned topics

Enter part of a topic name to get a list of similar topics, for example:
	* *faq* to get %SEARCH{ "faq" scope="topic" nosearch="on" nototal="on" format="[[$topic]]" separator=", " }%

__Note:__ The Go box is not a search box
