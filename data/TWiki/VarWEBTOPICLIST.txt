#VarWEBTOPICLIST
---+++ WEBTOPICLIST -- standard navigation links

This variable is intended to be used in skin templates to provide a
set of standard navigation links.
   * Type: Preference variable defined in [[%TWIKIWEB%.%WIKIPREFSTOPIC%]].  Can be customized for a TWiki site in [[%LOCALSITEPREFS%]], and for each web in their %WEBPREFSTOPIC% topics.
   * Current value: WEBTOPICLIST = %WEBTOPICLIST%
   * Related: TWikiTemplates
