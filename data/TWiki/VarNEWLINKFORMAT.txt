#VarNEWLINKFORMAT
---+++ NEWLINKFORMAT{"html"} -- Customize the new !WikiWord link
   * Syntax: =%<nop>NEWLINKFORMAT{"html"}%=
The =NEWLINKFORMAT= can contain the following variables:
   * $web: the web where the new topic is to be created.
   * $topic: the name of the new topic.
   * $text: the link text    
   
   * Example: =  * Set NEWLINKFORMAT = <a  class="twikiNewLink" href="%SCRIPTURLPATH{"edit"}%/$web/$topic" rel="nofollow" title="%MAKETEXT{"Create this topic"}%">$text</a>= 
   * Replaces: NEWTOPICLINKSYMBOL
