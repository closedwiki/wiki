---+++ SITEMAPLIST, SITEMAPWHAT, SITEMAPUSETO -- control [[%TWIKIWEB%.SiteMap]] appearance


The variables SITEMAPLIST, SITEMAPWHAT, and SITEMAPUSETO together
control the appearance of the [[%TWIKIWEB%.SiteMap][SiteMap]].  They
are initialized when a web is created (see %TWIKIWEB%.ManagingWebs),
stored in the %WEBPREFSTOPIC% topic of each web and used only in the
[[%TWIKIWEB%.SiteMap][SiteMap]].
   * For the webs which you want listed in the [[%TWIKIWEB%.SiteMap][SiteMap]], set SITEMAPLIST to =on=, do not set NOSEARCHALL, and supply values for SITEMAPWHAT and SITEMAPUSETO.
   * Type: Preference variable defined in [[%TWIKIWEB%.%WIKIPREFSTOPIC%]].  Can be customized for a TWiki site in [[%LOCALSITEPREFS%]], and for each web in their %WEBPREFSTOPIC% topics.
   * Related: [[%TWIKIWEB%.SiteMap]], %TWIKIWEB%.ManagingWebs
