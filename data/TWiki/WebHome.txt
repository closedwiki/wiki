---+++ TWiki Site Tools

Site tools include utilities for navigating and keeping up with site activity on individual TWiki webs. Preferences can also be configured, by web or site-wide. You are currently in the *%TWIKIWEB%* web.

<form action="%SCRIPTURLPATH%/search%SCRIPTSUFFIX%/%WEB%/">
	* WebSearch: Full text search of the %WEB% web<br>
	  <input type="text" name="search" size="32" /> <input type="submit" value="Search" /> &nbsp;&nbsp; (extended options in WebSearch)
	* WebChanges: Display recent input on the %WEB% web
	* WebIndex: List all %WEB% topics in alphabetical order
	* %NOTIFYTOPIC%: Subscribe to an email alert with the latest %WEB% web changes 
	* %STATISTICSTOPIC%: View access statistics for the %WEB% web
	* %WEBPREFSTOPIC%: Preference settings for the %WEB% web
	* %WIKIPREFSTOPIC%: %WIKITOOLNAME% site-wide preferences
</form>
