---+++ TWiki Site Tools

TWikiSiteTools include utilities for navigating and keeping up with site activity on individual TWiki webs. Preferences can also be configured, by web or site-wide. You are currently in the *%TWIKIWEB%* web.

<form action="%SCRIPTURLPATH%/search%SCRIPTSUFFIX%/%WEB%/">
	* WebSearch: Full text search of this web<br>
	  <input type="text" name="search" size="32" /> <input type="submit" value="Search" /> &nbsp;&nbsp; (extended options in WebSearch)
	* WebChanges: Display recent input on the this web
	* WebIndex: List all web topics in alphabetical order. See also the faster WebTopicList
	* %NOTIFYTOPIC%: Subscribe to an email alert with the latest this web changes 
	* %STATISTICSTOPIC%: View access statistics for the this web
</form>
	* %WEBPREFSTOPIC%: Preference settings for the this web
	* %WIKIPREFSTOPIC%: TWiki site-wide preferences
