#VarMAROON

---+++ MAROON -- preference value for colored text

 =MAROON= is one of the rendering shortcut settings
predefined in TWikiPreferences.  See the section
[[TWikiPreferences#Rendering_Shortcuts_Settings][Rendering Shortcut Settings]]
in that topic for a complete list of colors.

   * Syntax: =%<nop>MAROON% maroon text %<nop>ENDCOLOR%=
   * Expands to: %MAROON% maroon text %ENDCOLOR%
   * __Note:__ =%&lt;color&gt;%= text must end with =%<nop>ENDCOLOR%=. If you want to switch from one color to another one you first need to end the active color with =%<nop>ENDCOLOR%=, e.g. write =%<nop>RED% some text %<nop>ENDCOLOR% %<nop>GREEN% more text %<nop>ENDCOLOR%=.
   * Related: [[TWikiVariables#VarENDCOLOR][ENDCOLOR]], [[TWikiPreferences#Rendering_Shortcuts_Settings][TWikiPreferences]], StandardColors
