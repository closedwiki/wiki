---+!! TWiki Documentation Graphics and Filetype icons

This is the TWiki icon library. The graphics can be used in topics and by web applications. 

%TOC{title="Contents:"}%

---++ Usage

There are several ways to put an image in a topic.

*Shorthand notation:* <code>%<nop>ICON{help}%</code> results in: %ICON{help}%

*Really shorthand notation:* in %TWIKIWEB%.TWikiPreferences set a variable to an ICON, for example:
<verbatim>
   * Set H = %ICON{help}%
</verbatim>
Now you can use the icon by writing =%<nop>H%=.

There are other approaches as well:
   * In any topic, write =%<nop>PUBURL%/%<nop>TWIKIWEB%/%TOPIC%/help.gif= to show the %PUBURL%/%TWIKIWEB%/%TOPIC%/help.gif icon
   * To get the full icon filename path, call =ICONPATH=: <code>%<nop>ICONPATH{"toggleopen"}%</code> gets you: <code>%ICONPATH{"toggleopen"}%</code> (if you paste that in a topic you will get: %ICONPATH{"toggleopen"}%
   * The TWiki:Plugins/TopicVarsPlugin lets you set variables in any topic, not just in WebPreferences, for use in the same topic
   * The TWiki:Plugins/SmiliesPlugin, using a different short hand than =<nop>%VARIABLES%=, may be worth checking out for extended use

To create an image with a link, write: =[<nop>[WebHome][%<nop>ICON{home}%]]= to get: [[WebHome][%ICON{home}%]]
