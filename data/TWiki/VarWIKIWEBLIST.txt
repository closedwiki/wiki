#VarWIKIWEBLIST
---+++ WIKIWEBLIST -- static list of webs

List of <nop>%WIKITOOLNAME% webs - on the first line of the upper toolbar (to add a new web, see ManagingWebs and %TWIKIWEB%.SiteMap)

   * Type: Preference variable defined in [[%TWIKIWEB%.%WIKIPREFSTOPIC%]].  Can be customized for a TWiki site in [[%LOCALSITEPREFS%]], and for each web in their %WEBPREFSTOPIC% topics.
   * Current value: WIKIWEBLIST = %WIKIWEBLIST%
   * %T% *NOTE:* To automatically list all non-hidden webs in alphabetical order, use the =%<nop>WEBLIST{}%= variable instead: <br />
     %WEBLIST{"[[$name.%HOMETOPIC%][$name]]" separator=" | "}%
   * Related: [[TWikiVariables.#VarWEBLIST][WEBLIST]], TWikiTemplates
