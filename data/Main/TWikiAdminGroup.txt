---+++++ Setup Note:

   a. Edit this group topic page (click Edit)
   a. Enable the GROUP variable by removing the # sign
   a. Insert wanted %MAINWEB%.TWikiUsers in the TWiki Administrator Group by listing them in the GROUP variable %BR%
      (please remove the demonstration user 'ATWikiUser' as well)
   a. Save this topic
   a. Verify that new members show up properly in the group listing at %MAINWEB%.TWikiGroups
   a. Enable the ALLOWTOPICCHANGE variable by removing # sign 
   a. Save this topic again - <nop>TWikiAdminGroup is now locked to allow editing only by listed users in GROUP
   a. Suggestion: Add the ALLOWTOPICHANGE variable to %TWIKIWEB%.TWikiPreferences and set it to this group (<nop>TWikiAdminGroup), %BR%
      restricting edit of site-wide preferences to the TWiki Administrator Group
   a. Remove this setup note after you have finished setup
