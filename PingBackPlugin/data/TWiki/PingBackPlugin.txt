This implements a pingback service for TWiki. See the pingback specification at
http://hixie.ch/specs/pingback/pingback

There's a PingBackClient for testing. 

---++ Syntax Rules
=%<nop>PINGBACK{...}%=


---++ Plugin Settings

   * Set SHORTDESCRIPTION = Pingback service for TWiki
   * Set PINGBACKSERVER = http://jojowiki.dyndns.org/pingback
   * #Set PINGBACKSERVER = http://jojowiki.dyndns.org/rest/PingBackPlugin/server
