Converts an email address into a WikiWord. If you use email addresses as a login name, there are issues with @ and . characters. This plugin uses simple regular expressions to change an email address into well formatted alphanumerics.

If you want to allow TWikiRegistration so that the long email login can be mapped to a shorter WikiName, change the setting in =configure= to allow login name, and update the TWikiRegistration page to show !%WIKIUSER% as the default value instead of !%REMOTE_USER%.
