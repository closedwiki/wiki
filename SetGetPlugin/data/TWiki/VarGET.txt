#VarGET
---+++ GET{"name"} -- get a variable
   * Get the value of a named variable, previously set with =%<nop>SET{}%=.
   * Syntax: =%<nop>GET{"name"}%=
   * Example: =%<nop>GET{"lunch"}%= returns =Sushi= if this has been previously set: =%<nop>SET{"lunch" value="Sushi"}%=
