#VarSET
---+++ SET{"name" value="..."} -- set a variable
   * Set a named variable, can be retrieved later with =%<nop>GET{}%=.
   * Syntax: =%<nop>SET{"name" value="..."}%=
   * Example: =%<nop>SET{"lunch" value="Sushi"}%=
