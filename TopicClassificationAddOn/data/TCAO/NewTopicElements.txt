The following topic elements are displayed in a banner at the top of new !TopicTypes and !SubjectAreas when they are first created. It offers a list of likely follow-up actions and additional instructions for using the particular !TopicType or !SubjectArea.

*Note on creating new =INSTRUCTIONS= messages:* The INSTRUCTIONS variable must be defined in the template topic (taken from the %TCAOWEB%/Installs.WebHome web) for the !TopicType or !SubjectAreas. If there are no additional instructions, set it to "no". If there are additional instructions for installing or using the particular !TopicType, then define a section in this topic and set the INSTRUCTIONS variable to the name of that section.
