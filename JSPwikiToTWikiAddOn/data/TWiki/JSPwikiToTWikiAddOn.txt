---+ JSPwiki to TWiki Add-On

_converts JSPwiki-filestructure to a TWiki capable one._

there are some things that do not get converted correctly but it should be a
good starting point:

   * Attachements linked from other topics that contain spaces don't link
   * table headers can be messed up
   * the parent topic relation is lost
   * and maybe more

---++ Add-On Installation Instructions

__Note:__ You do not need to install anything on the browser to use this
add-on. The following instructions are for the administrator who installs the
add-on on the server where TWiki is running. 

   * edit marked variables in attached perl-script
   * run script

---++ Add-On Info

   * Set SHORTDESCRIPTION = Converts JSPwiki filestructure to TWiki file
   * format
