A slide style plugin for use on single slides.

%TOC%

------

---++ Syntax Rules

   * All slides start with a %<nop>SLIDESTART%  and end with a %<nop>SLIDEEND%
   * Slide styles are controlled through HTML tables stored in their own topic.
   * To utilized a particular style, supply the topic name as an argument to the %<nop>SLIDESTART% command like this:
<verbatim>
%<nop>SLIDESTART{ template="SlideTemplate" }%
</verbatim>
   * Slides should be layed out just like normal topics.  If you want a bulleted slide then use normal bulleting.
   * Comments come at the end of the slide and follow a <nop>---++ Comments second level header.  Everything after the comments header is treated as a comment.
   * To show or hide comments the <nop>SLIDEPLUGIN_HIDECOMMENTS will need to be modified.  Its default value is False.

---++ Plugin Settings

Plugin settings are stored as preferences variables. To reference a plugin setting write ==%<nop>&lt;plugin&gt;_&lt;setting&gt;%==, i.e. ==%<nop>INTERWIKIPLUGIN_SHORTDESCRIPTION%==

   * One line description, is shown in the %TWIKIWEB%.TextFormattingRules topic:
      * Set SHORTDESCRIPTION = PlugIn for Slide Presentation

   * Display Comments
      * Set SLIDEPLUGIN_HIDECOMMENTS = False
