---+ Static Method

A !StaticMethod is a method in a package that can be called without reference to an object of the package type. For example,

---+ package Telecoms
---++ !StaticMethod defaultColor()

<verbatim>
my $colour = Telecoms::defaultColor()
</verbatim>
