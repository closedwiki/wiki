#VarMASTERWEBSCRIPTURL
---+++ MASTERWEBSCRIPTURL{"script"} -- the script URL on the master web
   * Returns the URL of the specified script on the master site of the specified web.
   * Syntax: =%<nop>MASTERWEBSCRPITURL{"script" ...}%=
   * Supported parameters:
     | *Parameter:* | *Description:* | *Default:* |
     | ="script"= | The name of the script. | =view= |
     | =web= | The name of the web. | The current web |
   * Example: %<nop>MASTERWEBSCRIPTURL{"edit" web="Foo"}% -&gt; http:<nop>//twikimaster.example.com/cgi-bin/edit/Foo
   * Category: AdministrationVariables, SystemInformationVariables
