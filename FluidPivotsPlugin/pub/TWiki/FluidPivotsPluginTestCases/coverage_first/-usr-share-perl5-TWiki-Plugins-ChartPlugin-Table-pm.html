25#
26# Constructor
27# new($topicContents) - Create a 'Table' object from topic contents
28# Getters/Setters
29# checkTableExists($name) - Check if the specified table name exists
30# getTable($num) - Return the specified table
31# getTableInfo - DEBUG purposes only. Print out contents
32# of tables in table object
33# getTableRange($range) - Given a range of rows/columns in
34# SpreadSheetPlugin format, return an array
35# of staring row/column ending row/column
36# getRow($row,$c1,$c2) - Return the data at the specified row
37# starting at column 1 and ending at column 2
38# getData($tblnum,$range) - Return the data at the specified range.
39# If a single row or single column, then
40# return the data. If multiple
41# rows/columns, then return the data in row
42# format.
43# getRowColumnCount($range) - Return the number of rows/columns
44# specified in the range
45# getDataRows($tblnum,$range)- Return the data at the specified range
46# assuming that the data is in row format
47# NOTE: In the case of multiple
48# rows/columns, this is identical to
49# getData().
50# getDataColumns($tblnum,$range)- Return the data at the specified range
51# assuming that the data is in column format
