---+ Timeline TWiki Plugin

This plugin uses the DHTML-based AJAX widget for visualizing time-based events from http://simile.mit.edu/timeline/ to show any formatted chronological events. I use it to show <nop>WebChanges with the query below, and will update with a task / calendar query soon.

<img src="%ATTACHURLPATH%/timeline.jpg" alt="timeline.jpg" width='300' height='328' />

---++ Syntax Rules

<verbatim>
%TIMELINE{
"%SCRIPTURL{view}%/Sandbox/TimeLineSearch"
interval="DAY"
height="450px"
width="450px"
date="%DISPLAYTIME{"$day $month $year"}%"
}%
</verbatim>

where !TimeLineSearch conatins
<verbatim>
<data>
<!--
</verbatim>
<pre>
   * Set <nop>SKIN = xml
</pre>
<verbatim>

the format of and event:  
   start="May 28 2006 09:00:00 GMT"
   end="Jun 15 2006 09:00:00 GMT"
   isDuration="true" 
   title="Writing Timeline documentation" 
   image="http://simile.mit.edu/images/csail-logo.gif" 
-->
%SEARCH{
".*" web="%INCLUDINGWEB%" regex="on" nototal="on" noheader="on" nosearch="on" order="modified" reverse="on" limit="50"
format="<event start='$date' title='$topic' link='%SCRIPTURL{"view"}%/$web/$topic' >$summary(noheader)</event>"}%
</data>
</verbatim>

---++ Example

In the installation package of this plugin you will find a demonstration topic that will unzip to the Sandbox web. After installation steps are done, please visit the [[Sandbox.TimelinePluginVisualizationTopic]] to see a live example.

---++ Plugin Settings

Plugin settings are stored as preferences variables. To reference
a plugin setting write ==%<nop>&lt;plugin&gt;_&lt;setting&gt;%==, i.e. ==%<nop>TIMELINEPLUGIN_SHORTDESCRIPTION%==

   * One line description, is shown in the %TWIKIWEB%.TextFormattingRules topic:
      * Set SHORTDESCRIPTION = Show a scrolling timeline based on a URL containing a XML document with events. The events list can be constructed from a TWiki query.

   * Debug plugin: (See output in =data/debug.txt=)
      * Set DEBUG = 0

---++ Plugin Installation Instructions

   * Unzip the zip/tgz file into your twiki base directory
   * If you are running TWiki release 4.0.4 or before, you need to apply the code change and add the tmpl file mentioned in http://develop.twiki.org/~develop/cgi-bin/view/Bugs/Item2617
   * Enable the plugin using =configure=
