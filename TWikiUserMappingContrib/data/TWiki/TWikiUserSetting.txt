---+++ User Settings -- preference variables customizable by users

Some of the TWikiPreferences are explicitly intended to be customized by TWiki users, though a default is provided in [[%SYSTEMWEB%.%WIKIPREFSTOPIC%]], and a site customisation can take place in [[%LOCALSITEPREFS%]].

Users can customize these variables by assigning them values. Example:
<verbatim>
   * Set LINKTOOLTIPINFO = off
</verbatim>
