---+ <nop>%TOPIC%

Write links to javadoc classes

---++ Syntax Rules

	* Write the command =%<nop>JAVADOC{source:fullclassname}%= anywhere in your topics. It will be replaced by a link to the javadoc page.
	* The source refers to a preference variable setting the URL to the javadoc root
	* You can ommit the source and write =%<nop>JAVADOC{fullclassname}%=, in this case a default source is used (see settings below) 

---++ Examples

	| *You type...* | *You should see...* | *If installed...*|
	| =%<nop>JAVADOC{L4J:org.apache.log4j.Logger}%= |<A HREF="http://jakarta.apache.org/log4j/docs/api/org/apache/log4j/Logger.html">Logger</A>| %JAVADOC{L4J:org.apache.log4j.Logger}%|
	| =%<nop>JAVADOC{java.lang.String}%= |<A HREF="http://java.sun.com/j2se/1.4.1/docs/api/java/lang/String.html">String</A> | %JAVADOC{java.lang.String}%|
	| =%<nop>JAVADOC{IO:BufferedOutputStream}%= |<A HREF="http://java.sun.com/j2se/1.4.1/docs/api/java/io/BufferedOutputStream.html">BufferedOutputStream</A>| %JAVADOC{IO:BufferedOutputStream}%|

	In the last example, the source contain the package, so you need only to write the class name.

---++ <nop>%TOPIC% Global Settings

Plugin settings are stored as preferences variables. To reference a plugin setting write ==%<nop>&lt;plugin&gt;_&lt;setting&gt;%==, i.e. ==%<nop>INTERWIKIPLUGIN_SHORTDESCRIPTION%==

	* One line description, is shown in the %TWIKIWEB%.TextFormattingRules topic:
		* Set SHORTDESCRIPTION = Write links to javadoc classes

	* Debug plugin: (See output in =data/debug.txt=)
		* Set DEBUG = 0

	* javadoc sources
		* Set DEFAULT = http://java.sun.com/j2se/1.4.1/docs/api/
		* Set L4J = http://jakarta.apache.org/log4j/docs/api/
		* Set IO = http://java.sun.com/j2se/1.4.1/docs/api/java/io/
