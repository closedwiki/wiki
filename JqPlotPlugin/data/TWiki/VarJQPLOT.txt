#VarJQPLOT
---+++ JQPLOT -- add references to jqPlot !JavaScript library
   * [[http://www.jqplot.com/][jqPlot]] is a "pure javascript plotting" library. This variable is handled by the JqPlotPlugin to add the required !JavaScript and CSS tags to =&lt;head&gt;=.
   * Syntax: =%<nop>JQPLOT%=, =%<nop>JQPLOT{"jqPlot plugin name(s)"}%=
   * Example: =%<nop>JQPLOT{"barRenderer, pieRenderer"}%=
   * Related: JQueryPlugin
   * Category: ChartingAndDrawingVariables, ApplicationsAndComponentsVariables, DevelopmentVariables
